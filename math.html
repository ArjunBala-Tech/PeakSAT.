The Math Commandments

Commandment I: The Discriminant
If a quadratic has "no real solutions", set b² − 4ac < 0. If it has "one solution", set b² − 4ac = 0.
Commandment II: Circle Equations
Always factor (x² + ax) + (y² + by) = c into (x−h)² + (y−k)² = r² by completing the square.
Commandment III: System Traps
"Infinitely many solutions" means the lines are identical. Match their slopes and y-intercepts exactly.
